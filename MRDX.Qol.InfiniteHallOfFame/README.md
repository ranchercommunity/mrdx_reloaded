# Infinite Hall of Fame - v0.1.0

* Allows for save files to have unlimited Hall of Fame space.
* Select the Hall of Fame page by using left and right on the D-Pad when in the Hall of Fame monster selection menu.



### Hall of Fame File Notes

* HoF data is saved to the MR2DX save folder.
* The file contains a 96 byte header, followed by, in multiples of three, 96 byte sets representing the three HoF slots.
* HoF files can be transferred or backed up. Be sure to not overwrite existing hall of fame data.
